# <center><span style="color:#8280DC">Wrighth Rules</span></center>
## General
Pronounced Backwards
Never Resets
	Only Able To Be Removed Or Altered By A Wrighther Of Equal Or Greater Prowess
Pictograms Are Safer
One Syllable Per Glyph To Reduce Chance Of Reinterpretation 
Loop-De-Loop Arrow = Finish Loop And Wait For Trigger
No Loop-De-Loop Arrow = Continue Through
## Terminology
Wright = The System Of Spellcasting, Or The Finished Spell
Glyphs = The Individual Symbols
Nodes = The Symbol Space
Flow = The Order Of The Nodes
Branch = Divergent Paths
Circuit = A Closed  Flow Of Nodes And Branches
Prominence = How Much Certain Aspects Stand Out
	Style = Details That Are Added Without Purpose
Ingredients = Material Components Added To The Wrighth
Expandability = How Easily Something Could Be Added To The Wrighth
Chargeable Vs Ironclad = If The Values Within The Nodes Are Set In Stone Or Not
(In)definity = Defined Wrighths End Instead Of Looping
(Pre)condition = The Trigger For The Node
Enhancement = Leeway Or Stipulation 
Clarification = How The Wrighth Is To Activated
Timing = Rate, Speed, Or Pacing
Unsolved ("Hidden") = If It Can Be Seen
Context = Something That Gets Established
